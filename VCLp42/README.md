# 📝 Ứng dụng Quản lý Việc Cần Làm (VCLp4)

Ứng dụng Android quản lý công việc thời gian thực sử dụng **Firebase Realtime Database** và **Firebase Authentication**. Dự án được xây dựng với giao diện **Material Design 3** hiện đại, chuyên nghiệp.

## 🚀 Tính năng chính
- **Đăng nhập/Đăng ký**: Hệ thống xác thực an toàn qua Email/Password.
- **Ghi nhớ đăng nhập**: Tự động vào thẳng ứng dụng nếu đã đăng nhập trước đó.
- **Quản lý công việc**: Thêm, lưu trữ và đồng bộ hóa công việc thời gian thực.
- **Giao diện cao cấp**: Sử dụng CardView, Vector Assets và hệ màu sắc hài hòa.

## 🛠 Công nghệ sử dụng
- **Ngôn ngữ**: Java
- **UI/UX**: Material Design 3, ConstraintLayout, RecyclerView
- **Backend**: Firebase Authentication, Firebase Realtime Database
- **Build System**: Gradle (Kotlin DSL) với Version Catalog

## ⚙️ Cấu hình dự án
1. Clone dự án về máy.
2. Thêm tệp `google-services.json` vào thư mục `app/`.
3. Mở dự án bằng Android Studio.
4. Chỉnh sửa `compileSdk` và `targetSdk` nếu cần thiết (Hiện tại là 35/36).
5. Build và chạy ứng dụng.
